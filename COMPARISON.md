## Adonis

`npm init adonis-ts-app@latest <app-name>`

### To Start Dev Server

`node ace serve --watch`

## Sails

```bash
> npm i -g sails
> sails new <app-name>
```

### To Start Dev Server

`sails lift`

## Nest

```bash
> npm i -g @nestjs/cli
> nest new <app-name>
```

### To Start Dev Server

`npm run start`
